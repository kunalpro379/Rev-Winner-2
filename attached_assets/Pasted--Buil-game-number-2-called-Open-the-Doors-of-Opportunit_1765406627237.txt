 Buil game number 2 called 'Open the Doors of Opportunities'

Objective

Build an interactive, animated sales quiz game for lead generation and engagement. The game should be hosted inside the Rev Winner website and accessible as a dedicated menu item called “Sales Challenge.”
The Sales Challenge page must display both:

1. The existing game


2. This new Rev Winner game
'Open the Doors of Opportunities'


Users should be able to select which game they want to play.


---

1. Entry Flow and Lead Generation Form

When a user clicks on Sales Challenge from the main website menu:

1. The user first lands on the Sales Challenge page.


2. This page shows:

The earlier existing game

The new Rev Winner Sales Challenge game



3. When the user clicks on the new Sales Challenge game, they are shown a lead capture form.



Lead Capture Form Fields

Full Name

Company Name

Email Address


This form is mandatory before entering the game.
All submitted data should be stored securely for lead generation.

After form submission, the user enters the game.


---

2. Main Game Concept

The user plays as a sales professional represented by a 3D animated Owl character in a professional suit.

The owl must open 10 sequential doors by answering 10 customer-style questions correctly.

Each door represents one stage of a real-world sales conversation.


---

3. Visual Layout and UI Placement

Screen Structure

Top Center or Top Right:

Displayed in a dialogue bubble style

This customer visually asks each question.


Middle of Screen:
The Question Text Area

Styled like a real sales conversation


Bottom Section:
The Answer Options Panel

Clearly spaced and visually highlighted


Top Corner:
Live Score Indicator

Shows:

Current Score

4. Question Logic and Personalization

All questions must be dynamically generated based on the company name entered in the form.

Question Types

Product-based questions

Service-based questions

Technical capability questions

Sales discovery and objection-handling questions


Sample Question Logic

If Company = ServiceNow
Question:
"What does ITSM stand for in ServiceNow?"

Options:

Information Technology Service Management ✅

International Technology Sales Model ❌

Integrated Technical Support Machine ❌

Infrastructure Tracking Service Method ❌


If Company = Microsoft Azure
Question:
"Which service is used to host virtual machines in Azure?"

Options:

Azure Virtual Machines ✅

Azure DevOps ❌

Azure Logic Apps ❌

Azure Functions ❌

5. Answer Behavior and Animation

If Answer is Correct

Owl character:

Shows confident body movement

Slight victory gesture

Door opens smoothly with animation


Score increases

Next question loads immediately


If Answer is Incorrect

Owl character:

Shows mild disappointment animation


Display message: "No worries. Sales is always a learning process."

Show Rev Winner Demo Booking Button

End the game immediately
