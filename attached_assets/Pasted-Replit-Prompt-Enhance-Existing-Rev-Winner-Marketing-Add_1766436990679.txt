Replit Prompt: Enhance Existing Rev Winner Marketing Add-On
Objective
Enhance the existing Rev Winner Marketing Add-On (already built and accessible via a hidden page, not visible in the main menu).
This task is not to create a new add-on, but to extend and modify the existing one with advanced AI-driven marketing capabilities using stored conversation data and universal knowledge.
All enhancements must remain contained within the existing marketing add-on page and must not interfere with any other Rev Winner features.
Core Data Principles
All customer and user conversations already stored in Rev Winner act as the Data Bank.
Every user’s conversation history must be stored in one centralized location and accessible for content generation.
Content generation must clearly follow data-source rules defined by user-selected toggles.
